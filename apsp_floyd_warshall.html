Floyd-Warshall

Θ(n³)
Complessità temporale
Teoria
Floyd-Warshall definisce D(k)[i][j] come il peso del cammino minimo da i a j i cui vertici intermedi appartengono all'insieme {1,…,k}.
D⁽⁰⁾[i][j] = W[i][j]
Il passo ricorsivo considera se conviene passare per il vertice k:
D⁽ᵏ⁾[i][j] = min( D⁽ᵏ⁻¹⁾[i][j], D⁽ᵏ⁻¹⁾[i][k] + D⁽ᵏ⁻¹⁾[k][j] )
Π[i][j] viene aggiornato solo quando si trova un cammino migliorativo tramite k. Per rilevare cicli negativi: se D[i][i] < 0 per qualche i, il grafo ne contiene uno.

Pseudocodice
FLOYD-WARSHALL(W): n = W.rows D = W // D⁽⁰⁾ = W Π[i][j] = i se W[i][j] < ∞ for k = 1 to n: for i = 1 to n: for j = 1 to n: if D[i][j] > D[i][k] + D[k][j]: D[i][j] = D[i][k] + D[k][j] Π[i][j] = Π[k][j] // predecessore return D, Π
