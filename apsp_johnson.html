Teoria · 4 fasi
Johnson è ottimale per grafi sparsi. Gestisce pesi negativi senza cicli negativi combinando Bellman-Ford e Dijkstra.
1. Aggiunge vertice fittizio s collegato a tutti i nodi con peso 0.
2. Esegue Bellman-Ford da s per ottenere potenziali h(v).
3. Ri-pesatura degli archi:
ŵ(u,v) = w(u,v) + h(u) − h(v) ≥ 0
4. Dijkstra da ogni vertice, poi correzione:
d(u,v) = d̂(u,v) − h(u) + h(v)

JOHNSON(G, w): G'= G + s, archi (s,v,0) ∀v if BELLMAN-FORD(G',w,s) = FALSE: return "ciclo negativo" h(v) = δ(s,v) // da B-F for (u,v) ∈ E: ŵ(u,v) = w(u,v)+h(u)−h(v) for u ∈ V: d̂[u][·] = DIJKSTRA(G, ŵ, u) D[u][v] = d̂[u][v]−h(u)+h(v) return D, Π
